# e-plantShopping 🌿

## Paradise Nursery Shopping Cart Application

This project is a React-based shopping cart application for an online plant shop called **Paradise Nursery**. It allows users to browse different categories of houseplants, add items to a cart, and manage their selections dynamically.

---

## 🌱 Project Overview

The **e-plantShopping** application simulates a real-world e-commerce experience for plant lovers. Users can:

- Browse a variety of houseplants grouped into categories
- View plant details such as image, description, and price
- Add plants to a shopping cart
- Increase or decrease item quantities
- Remove items from the cart
- View total cost updates in real time

---

## 🚀 Features

### 🏠 Landing Page
- Displays the Paradise Nursery brand
- Includes a "Get Started" button to navigate to the product page

### 🌿 Product Listing Page
- Displays multiple plant categories (e.g., Aromatic, Medicinal, Indoor Plants)
- Each plant includes:
  - Image
  - Name
  - Description
  - Price
  - Add to Cart button
- Add to Cart button becomes disabled after adding
- Dynamic cart icon showing total items

### 🛒 Shopping Cart Page
- Displays all selected items
- Shows:
  - Plant image, name, and unit price
  - Quantity controls (increase/decrease)
  - Subtotal per item
  - Total cart amount
- Includes:
  - Delete button
  - Continue Shopping button
  - Checkout button ("Coming Soon")
